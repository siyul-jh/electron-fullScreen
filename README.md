# Electron FullScreen

PC에 특정 프로그램으로 인해 브라우저의 전체화면 제약이 걸렸다.

따라서 Youtbue와 같은 동영상을 전체화면으로 볼 수 없었다.

Window, Mac 환경은 다르기에 각각 OS에서 Build 해야한다.
